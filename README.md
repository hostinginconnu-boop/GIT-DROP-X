## 📋 Overview

**GitDrop X Premium** is an elegant and professional web application that simplifies creating GitHub repositories and automatically uploading files. With a modern, intuitive interface, you can manage your GitHub projects quickly and efficiently.

### ✨ Key Features

- **🔐 Secure Authentication**: Native GitHub integration using Personal Access Tokens
- **📦 Multiple File Upload**: Drag-and-drop support for easy file sending
- **🌍 Visibility Control**: Create public or private repositories as needed
- **⚡ Instant Deployment**: Create and deploy repositories in seconds
- **📊 Real-Time Console**: Track each step of the deployment process
- **🎨 Premium Interface**: Modern design with glassmorphism and smooth animations
- **🌐 Multilingual Support**: Available in English and French
- **📱 Responsive**: Works flawlessly on desktop, tablet, and mobile
- **⚙️ Session Statistics**: Track created repositories, uploaded files, and commits made

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- An active GitHub account
- A GitHub Personal Access Token with `repo` permissions

### Obtaining Your GitHub Token

1. Go to [github.com/settings/tokens/new](https://github.com/settings/tokens/new)
2. Give your token a name (e.g., "GitDrop X")
3. Select the **`repo`** scope (full control of private and public repositories)
4. Click **"Generate token"**
5. Copy the generated token (it won't be shown again)
6. Paste the token into the GitDrop X application

### Basic Usage

1. **Access the Application**: Open GitDrop X Premium in your browser
2. **Enter Your Token**: Paste your GitHub Personal Access Token in the input field
3. **Configure the Repository**:
   - Set the repository name
   - Choose whether it will be public or private
   - Add a description (optional)
   - Customize the commit message
4. **Add Files**: Drag and drop files or click to select them
5. **Deploy**: Click the "CREATE & DEPLOY ⚡" button
6. **Monitor**: Watch the real-time progress in the console
